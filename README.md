# Projeto Análise Preditiva com AI

# Sobre o Projeto

Aplicativo de análise de ações para predições com Machine Learning. 
Proposta de projeto do curso da Daxus [Empowerdata Treinamentos](https://www.empowerdata.com.br/). 

# Tecnologias utilizadas

- Python
- Jupyter Notebook
- Pandas
- yfinance
- Prophet
